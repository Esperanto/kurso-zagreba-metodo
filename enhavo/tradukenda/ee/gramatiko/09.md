# Ìpò àdéhùn

Ìpò àdéhùn ni a fi ìparí ọ̀rọ̀-ìṣe *__-us__* hàn:

- *mi kred__us__* (èmi yóò gbàgbọ́ / èmi bá gbàgbọ́)
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* (Bí ara mi bá dáa, inú mi yóò dùn gan-an.)

# *Kvazaŭ*

*Kvazaŭ* jẹ́ asopọ̀ tó túmọ̀ sí “bí ẹni pé”. Ó sábà ń tẹ̀lé ìpò àdéhùn.

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* (O jókòó níbẹ̀ bí ẹni pé ìwọ jẹ́ olówó.)

A tún lè lo ó ní fọ́ọ̀mù kúkúrú:

- *Vi sidas tie __kvazaŭ__ riĉulo.* (O jókòó níbẹ̀ bí olówó.)

# Àfikún *-ad*

*__-ad__* lè sọ ìṣe di orúkọ ìṣe:

- *kanti* (kọrin) → *kant__ad__o* (kíkọrin)
- *suferi* (jìyà) → *sufer__ad__o* (ìjìyà)

Ó tún lè fi ìṣe tó ń tẹ̀síwájú tàbí tó ń tún ṣe hàn:

- *rigardi* (wo) → *rigard__ad__i* (máa wò fún ìgbà díẹ̀ / tẹ̀síwájú ní wíwo)
- *demandi* (béèrè) → *demand__ad__i* (máa béèrè léraléra)
- *informo* (ìwífúnni) → *inform__ad__o* (ìpolongo)

# Àfikún *-ar*

*__-ar__* fi akojọpọ̀ ohun hàn bí ẹyọ kan:

- *arbo* (igi) → *arb__ar__o* (igbo)
- *vagono* (kẹ̀kẹ́ inú ọkọ̀ ojú irin) → *vagon__ar__o* (ọkọ̀ ojú irin)
- *vorto* (ọ̀rọ̀) → *vort__ar__o* (ìwé atúmọ̀ọ̀rọ̀)

# Àfikún *-um*

*__-um__* jẹ́ àfikún aláìtumọ̀ pàtó. Ìtumọ̀ rẹ̀ dá lórí ọ̀rọ̀ kọọkan:

- *plena* (kún) → *plen__um__i* (mú ṣẹ)
- *proksima* (sunmọ́) → *proksim__um__e* (ní ìsúnmọ́ iye)
- *suno* (òrùn) → *sun__um__i* (jókòó/ṣeré ní oorun)
- *malvarm__um__i* (ní òtútù ara)

**💡 Ìránnilétí:** Kọ́ àwọn ọ̀rọ̀ *-um* gẹ́gẹ́ bí ọ̀rọ̀ tuntun; wọn kì í tẹ̀lé ìlànà kan ṣoṣo.
