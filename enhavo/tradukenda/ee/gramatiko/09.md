# Ìpò àdéhùn

Ìpò àdéhùn ni a fi ìparí ọ̀rọ̀-ìṣe _**-us**_ hàn:

- _mi kred**us**_ (èmi yóò gbàgbọ́ / èmi bá gbàgbọ́)
- _Se mi est**us** sana, mi est**us** tre feliĉa._ (Bí ara mi bá dáa, inú mi yóò dùn gan-an.)

# *Kvazaŭ*

*Kvazaŭ* jẹ́ asopọ̀ tó túmọ̀ sí “bí ẹni pé”. Ó sábà ń tẹ̀lé ìpò àdéhùn.

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ (O jókòó níbẹ̀ bí ẹni pé ìwọ jẹ́ olówó.)

A tún lè lo ó ní fọ́ọ̀mù kúkúrú:

- _Vi sidas tie **kvazaŭ** riĉulo._ (O jókòó níbẹ̀ bí olówó.)

# Àfikún *-ad*

_**-ad**_ lè sọ ìṣe di orúkọ ìṣe:

- *kanti* (kọrin) → _kant**ad**o_ (kíkọrin)
- *suferi* (jìyà) → _sufer**ad**o_ (ìjìyà)

Ó tún lè fi ìṣe tó ń tẹ̀síwájú tàbí tó ń tún ṣe hàn:

- *rigardi* (wo) → _rigard**ad**i_ (máa wò fún ìgbà díẹ̀ / tẹ̀síwájú ní wíwo)
- *demandi* (béèrè) → _demand**ad**i_ (máa béèrè léraléra)
- *informo* (ìwífúnni) → _inform**ad**o_ (ìpolongo)

# Àfikún *-ar*

_**-ar**_ fi akojọpọ̀ ohun hàn bí ẹyọ kan:

- *arbo* (igi) → _arb**ar**o_ (igbo)
- *vagono* (kẹ̀kẹ́ inú ọkọ̀ ojú irin) → _vagon**ar**o_ (ọkọ̀ ojú irin)
- *vorto* (ọ̀rọ̀) → _vort**ar**o_ (ìwé atúmọ̀ọ̀rọ̀)

# Àfikún *-um*

_**-um**_ jẹ́ àfikún aláìtumọ̀ pàtó. Ìtumọ̀ rẹ̀ dá lórí ọ̀rọ̀ kọọkan:

- *plena* (kún) → _plen**um**i_ (mú ṣẹ)
- *proksima* (sunmọ́) → _proksim**um**e_ (ní ìsúnmọ́ iye)
- *suno* (òrùn) → _sun**um**i_ (jókòó/ṣeré ní oorun)
- _malvarm**um**i_ (ní òtútù ara)

**💡 Ìránnilétí:** Kọ́ àwọn ọ̀rọ̀ *-um* gẹ́gẹ́ bí ọ̀rọ̀ tuntun; wọn kì í tẹ̀lé ìlànà kan ṣoṣo.
