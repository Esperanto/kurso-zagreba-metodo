# *Ju … des*

*Ju … des* fi ìbáṣepọ̀ “bí ó ṣe … bẹ́ẹ̀ ni ó ṣe …” hàn.

- *Ju pli longe, des pli bone.* (Bí ó ṣe pẹ́ tó, bẹ́ẹ̀ ni ó ṣe dára tó.)

# *Ajn*

*Ajn* fi “eyikeyi” tàbí “bí ó ti wù kí ó rí” hàn lẹ́yìn tabelvorto:

- *kiom ajn* (bí iye tó wù kí ó jẹ́)
- *kie ajn* (níbo tó wù kí ó jẹ́)
- *kiam ajn* (nígbàkigbà)

# Ìṣáájú *dis-*

_**dis-**_ fi pínyà tàbí túká hàn:

- _**dis**ĵeti_ (tú ká / sọ ká)
- _**dis**sendi_ (tan kaakiri)

# Àfikún *-on*

_**-on**_ fi ìdá kan nínú odidi hàn:

- _du**on**o_ (ìdajì)
- _tri**on**o_ (ìdá mẹ́ta kan)
- _kvar**on**o_ (ìdá mẹ́rin kan)
- _ses**on**o_ (ìdá mẹ́fà kan)
