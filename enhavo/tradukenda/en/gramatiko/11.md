# The Suffix *__-il__*

forms words for a tool, implement or means:

- *tranĉ__il__o* – knife
- *hak__il__o* – axe
- *ŝlos__il__o* – key
- *timig__il__o* – scarecrow

**💡 Memory aid:** *-il* names the instrument for an action:

- *tranĉi* (to cut) → *tranĉilo* (knife)
- *ŝlosi* (to lock) → *ŝlosilo* (key)
