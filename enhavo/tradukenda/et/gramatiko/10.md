# Järelliide _**-an**_

_**-an**_ näitab liiget, elanikku või gruppi kuuluvat inimest:

- _klub**an**o_ - klubiliige
- _amerik**an**o_ - ameeriklane
- _samland**an**o_ - kaasmaalane

# Järelliide _**-ec**_

_**-ec**_ teeb abstraktse omaduse või seisundi:

- _bel**ec**o_ - ilu
- _varm**ec**o_ - soojus
- _neŭtral**ec**o_ - neutraalsus
- _ebl**ec**o_ - võimalus

# Järelliide _**-uj**_

_**-uj**_ võib tähendada anumat, puud või traditsioonilist maanime:

- _mon**uj**o_ - rahakott
- _pom**uj**o_ - õunapuu
- _Franc**uj**o_ - Prantsusmaa

# _**-ig**_ ja _**-iĝ**_ uuesti

- _**-ig**_ = midagi muutma või põhjustama
- _**-iĝ**_ = muutuma

Võrdle:

- *nova* → _nov**ig**i_ - uuendama, uueks tegema
- *fino* → _fin**iĝ**i_ - lõppema
- *informi* → _inform**iĝ**i_ - infot saama

# Tuttavad juured liitsõnades

Harjutustes korduvad tuttavad juured:

- *rilato* + *-e* → *tiurilate* - selles suhtes
- *parto* + *preni* → *partopreni* - osa võtma
- *sama* + *tempo* + *-e* → *samtempe* - samal ajal
- *plej* + *parto* + *-e* → *plejparte* - enamasti
