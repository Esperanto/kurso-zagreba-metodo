# *Ju ... des*

*ju pli ... des pli* tähendab "mida rohkem ..., seda rohkem ...":

- *Ju pli longe, des pli bone.* - Mida kauem, seda parem.

**💡 Mäluabi:** *ju* on esimeses pooles, *des* teises pooles.

# *Ajn*

*ajn* teeb tabelsõna vabaks või umbmääraseks. See seisab tabelsõna järel:

- *kie ajn* - kus iganes
- *kiam ajn* - millal iganes
- *iu ajn* - ükskõik kes

Vaata ka [tabelsõnu](/et/tabelvortoj/).

# Eesliide _**dis-**_

_**dis-**_ näitab lahku, laiali või eri suundades:

- _**dis**doni_ - laiali jagama
- _**dis**sendi_ - laiali saatma
- _**dis**ĵeti_ - laiali viskama

# Järelliide _**-on**_

_**-on**_ näitab murdosa:

- _du**on**o_ - pool
- _tri**on**o_ - kolmandik
- _kvar**on**o_ - veerand
- _dek**on**o_ - kümnendik

Sellest saab teha ka teisi vorme:

- *kvarona jaro* - veerand aastat
- *duonigi* - pooleks tegema

# Lõpu kordamine

Kursuse lõpus tulevad paljud vanad osad uutes sõnades tagasi:

- *trinki* (jooma) → *trinkaĵo* (jook)
- *mal* + *fermi* → *malfermi* (avama)
- *ordo* + *-ig* → *ordigi* (korda seadma)
- *ĉiu* + *tago* + *-e* → *ĉiutage* (iga päev)
- *re* + *veni* → *reveni* (tagasi tulema)
