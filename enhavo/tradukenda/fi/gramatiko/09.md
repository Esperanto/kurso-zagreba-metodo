# Konditionaali

Konditionaali ilmaistaan verbipäätteellä _**-us**_. Sitä käytetään hypoteettisissa tilanteissa, joissa suomessa olisi usein "-isi":

- _mi kred**us**_ - uskoisin, ajattelisin
- _Se mi est**us** sana, mi est**us** tre feliĉa._ - Jos olisin terve, olisin hyvin onnellinen.

**⚠️ Huomaa:** lauseen molemmat osat saavat *-us*-päätteen: sekä "jos"-lause että päälause.

**💡 Muistivihje:** verbijärjestelmä on nyt lähes kokonainen: *-i* (infinitiivi), *-as / -is / -os* (kolme aikamuotoa), *-u* (käsky/toive), *-us* (konditionaali). Kuusi päätettä riittää kaikille verbeille.

# *Kvazaŭ*

on konjunktio, jonka jälkeen käytetään tavallisesti konditionaalia:

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ - Istut siellä ikään kuin olisit rikas mies.

Sitä voidaan käyttää myös lyhyemmin:

- _Vi sidas tie **kvazaŭ** riĉulo._ - Istut siellä kuin rikas.

# Suffiksi _**-ad**_

muodostaa verbistä substantiivin, joka nimeää itse toiminnan:

- *kanti* (laulaa) → _kant**ad**o_ - laulaminen
- *suferi* (kärsiä) → _sufer**ad**o_ - kärsiminen

Se voi ilmaista myös jatkuvaa tai toistuvaa toimintaa:

- *rigardi* (katsoa) → _rigard**ad**i_ - katsella, katsoa pitkään
- *demandi* (kysyä) → _demand**ad**i_ - kysellä jatkuvasti
- *informo* (tieto) → _inform**ad**o_ - tiedotus

# Suffiksi _**-ar**_

ilmaisee kokoelmaa tai joukkoa kokonaisuutena:

- *arbo* (puu) → _arb**ar**o_ - metsä
- *vagono* (vaunu) → _vagon**ar**o_ - juna
- *vorto* (sana) → _vort**ar**o_ - sanakirja

**💡 Muistivihje:** *-ar* kokoaa yksittäiset asiat joukoksi:

- *arbo* (puu) → *arbaro* (metsä)

# Suffiksi _**-um**_

on yleissuffiksi, jolla ei ole yhtä kiinteää merkitystä:

- *plena* (täysi) → _plen**um**i_ - täyttää, toteuttaa
- *proksima* (läheinen) → _proksim**um**e_ - suunnilleen
- *suno* (aurinko) → _sun**um**i_ - ottaa aurinkoa
- _malvarm**um**i_ - vilustua

**💡 Muistivihje:** *-um* on suffiksien jokeri, kuten *je* ([oppitunti 6](/fi/06/gramatiko/)) on prepositioiden jokeri. Opettele *-um*-sanat yksitellen.
