# Konditionaali

Konditionaali ilmaistaan verbipäätteellä *__-us__*. Sitä käytetään hypoteettisissa tilanteissa, joissa suomessa olisi usein "-isi":

- *mi kred__us__* - uskoisin, ajattelisin
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* - Jos olisin terve, olisin hyvin onnellinen.

**⚠️ Huomaa:** lauseen molemmat osat saavat *-us*-päätteen: sekä "jos"-lause että päälause.

**💡 Muistivihje:** verbijärjestelmä on nyt lähes kokonainen: *-i* (infinitiivi), *-as / -is / -os* (kolme aikamuotoa), *-u* (käsky/toive), *-us* (konditionaali). Kuusi päätettä riittää kaikille verbeille.

# *Kvazaŭ*

on konjunktio, jonka jälkeen käytetään tavallisesti konditionaalia:

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* - Istut siellä ikään kuin olisit rikas mies.

Sitä voidaan käyttää myös lyhyemmin:

- *Vi sidas tie __kvazaŭ__ riĉulo.* - Istut siellä kuin rikas.

# Suffiksi *__-ad__*

muodostaa verbistä substantiivin, joka nimeää itse toiminnan:

- *kanti* (laulaa) → *kant__ad__o* - laulaminen
- *suferi* (kärsiä) → *sufer__ad__o* - kärsiminen

Se voi ilmaista myös jatkuvaa tai toistuvaa toimintaa:

- *rigardi* (katsoa) → *rigard__ad__i* - katsella, katsoa pitkään
- *demandi* (kysyä) → *demand__ad__i* - kysellä jatkuvasti
- *informo* (tieto) → *inform__ad__o* - tiedotus

# Suffiksi *__-ar__*

ilmaisee kokoelmaa tai joukkoa kokonaisuutena:

- *arbo* (puu) → *arb__ar__o* - metsä
- *vagono* (vaunu) → *vagon__ar__o* - juna
- *vorto* (sana) → *vort__ar__o* - sanakirja

**💡 Muistivihje:** *-ar* kokoaa yksittäiset asiat joukoksi:

- *arbo* (puu) → *arbaro* (metsä)

# Suffiksi *__-um__*

on yleissuffiksi, jolla ei ole yhtä kiinteää merkitystä:

- *plena* (täysi) → *plen__um__i* - täyttää, toteuttaa
- *proksima* (läheinen) → *proksim__um__e* - suunnilleen
- *suno* (aurinko) → *sun__um__i* - ottaa aurinkoa
- *malvarm__um__i* - vilustua

**💡 Muistivihje:** *-um* on suffiksien jokeri, kuten *je* ([oppitunti 6](/fi/06/gramatiko/)) on prepositioiden jokeri. Opettele *-um*-sanat yksitellen.
