# Sufiks _**-il**_

_**-il**_ označava alat, sredstvo ili pomagalo za neku radnju.

- *tranĉi* (rezati) → _tranĉ**il**o_ (nož)
- *haki* (sjeći) → _hak**il**o_ (sjekira)
- *skribi* (pisati) → _skrib**il**o_ (pisalo)
- *timigi* (plašiti) → _timig**il**o_ (strašilo)

**💡 Pomoć za pamćenje:** _**-il**_ odgovara pitanju "čime se to radi?".

# Sufiks _**-estr**_

_**-estr**_ označava voditelja, čelnika ili odgovornu osobu.

- *urbo* (grad) → _urb**estr**o_ (gradonačelnik)
- *ŝipo* (brod) → _ŝip**estr**o_ (kapetan broda)
- *lernejo* (škola) → _lernej**estr**o_ (ravnatelj škole)

U kombinaciji s _**-ar**_ može označiti vodeću skupinu.

- *urbo* (grad) + _**-estr**_ + _**-ar**_ → *urbestraro* (gradska uprava)

# Kombiniranje sufiksa

Poznati sufiksi mogu se pojaviti zajedno u jednoj riječi.

- *kuiri* (kuhati) + _**-il**_ + _**-ar**_ + _**-o**_ → *kuirilaro* (kuhinjski pribor)
- *pura* (čist) + _**-ig**_ + _**-il**_ + _**-o**_ → *purigilo* (sredstvo za čišćenje)
- *luma* (svijetao) + _**-ig**_ + _**-il**_ + _**-o**_ → *lumigilo* (rasvjetno sredstvo)

**⚠️ Pripazite:** čitaj sufikse redom. Svaki novi dio mijenja riječ koju si već dobio.

# Vrijeme, strana i točka kao dijelovi riječi

Korijeni *mez*, *tag*, *nokt*, *flank* i *punkt* grade vremenske i odnosne izraze. Zadnji korijen obično nosi glavno značenje.

- *mezo* (sredina) + *vojo* (put) + _**-e**_ → *mezvoje* (na pola puta)
- *tago* (dan) + *fino* (kraj) + _**-e**_ → *tagfine* (na kraju dana)
- *nokto* (noć) + *labori* (raditi) → *noktlabori* (raditi noću)
- *ĉefa* (glavni) + *punkto* (točka) → *ĉefpunkto* (glavna točka)
- *sama* (isti) + *flanko* (strana) + _**-e**_ → *samflanke* (na istoj strani)

**💡 Pomoć za pamćenje:** _**-e**_ na kraju često pretvara cijelu složenicu u priložni izraz, na primjer za vrijeme, mjesto ili stranu.
