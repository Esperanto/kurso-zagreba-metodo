# លក្ខខណ្ឌ

របៀបលក្ខខណ្ឌប្រើបញ្ចប់ *__-us__*។ វាប្រាប់អំពីអ្វីដែល “នឹង/អាចនឹង” កើតឡើងក្រោមលក្ខខណ្ឌមួយ។

- *mi kred__us__* – ខ្ញុំនឹងជឿ / ខ្ញុំគិតថា
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* – ប្រសិនបើខ្ញុំមានសុខភាពល្អ ខ្ញុំនឹងសប្បាយណាស់។

# *Kvazaŭ*

*kvazaŭ* មានន័យថា “ដូចជា” ហើយជាញឹកញាប់តាមដោយកិរិយាលក្ខខណ្ឌ៖

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* – អ្នកអង្គុយនៅទីនោះដូចជាអ្នកជាមនុស្សមាន។

វាក៏អាចប្រើខ្លីដោយគ្មានកិរិយាទាំងមូលបានដែរ៖

- *Vi sidas tie __kvazaŭ__ riĉulo.* – អ្នកអង្គុយនៅទីនោះដូចជាមនុស្សមាន។

# បច្ច័យ *-ad*

បច្ច័យ *__-ad__* បង្កើតនាមសកម្មភាព ឬបង្ហាញសកម្មភាពដែលបន្ត/ធ្វើម្តងហើយម្តងទៀត៖

- *kanti* (ច្រៀង) → *kant__ad__o* (ការច្រៀង)
- *suferi* (រងទុក្ខ) → *sufer__ad__o* (ការរងទុក្ខ)

សម្រាប់សកម្មភាពបន្ត៖

- *rigardi* (មើល) → *rigard__ad__i* (មើលបន្ត / សម្លឹង)
- *demandi* (សួរ) → *demand__ad__i* (សួរម្តងហើយម្តងទៀត)
- *informo* (ព័ត៌មាន) → *inform__ad__o* (ការផ្សព្វផ្សាយព័ត៌មាន)

# បច្ច័យ *-ar*

បច្ច័យ *__-ar__* បង្ហាញក្រុម ឬសំណុំដែលមើលជារួម៖

- *arbo* (ដើមឈើ) → *arb__ar__o* (ព្រៃ)
- *vagono* (ទូរថភ្លើង) → *vagon__ar__o* (រថភ្លើង)
- *vorto* (ពាក្យ) → *vort__ar__o* (វចនានុក្រម)

# បច្ច័យ *-um*

បច្ច័យ *__-um__* គ្មានន័យថេរតែមួយទេ។ ត្រូវរៀនពាក្យដែលមានវាជាពាក្យរៀងៗខ្លួន។

- *plena* (ពេញ) → *plen__um__i* (បំពេញ / សម្រេច)
- *proksima* (ជិត) → *proksim__um__e* (ប្រហែល)
- *suno* (ព្រះអាទិត្យ) → *sun__um__i* (ហាលថ្ងៃ)
- *malvarm__um__i* – ផ្តាសាយ

**💡 ចងចាំ:** *-um* ដូចជាប្រអប់ពិសេស។ កុំទាយអត្ថន័យពេក ត្រូវពិនិត្យពាក្យពេញ។
