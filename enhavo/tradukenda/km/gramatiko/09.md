# លក្ខខណ្ឌ

របៀបលក្ខខណ្ឌប្រើបញ្ចប់ _**-us**_។ វាប្រាប់អំពីអ្វីដែល “នឹង/អាចនឹង” កើតឡើងក្រោមលក្ខខណ្ឌមួយ។

- _mi kred**us**_ – ខ្ញុំនឹងជឿ / ខ្ញុំគិតថា
- _Se mi est**us** sana, mi est**us** tre feliĉa._ – ប្រសិនបើខ្ញុំមានសុខភាពល្អ ខ្ញុំនឹងសប្បាយណាស់។

# *Kvazaŭ*

*kvazaŭ* មានន័យថា “ដូចជា” ហើយជាញឹកញាប់តាមដោយកិរិយាលក្ខខណ្ឌ៖

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ – អ្នកអង្គុយនៅទីនោះដូចជាអ្នកជាមនុស្សមាន។

វាក៏អាចប្រើខ្លីដោយគ្មានកិរិយាទាំងមូលបានដែរ៖

- _Vi sidas tie **kvazaŭ** riĉulo._ – អ្នកអង្គុយនៅទីនោះដូចជាមនុស្សមាន។

# បច្ច័យ *-ad*

បច្ច័យ _**-ad**_ បង្កើតនាមសកម្មភាព ឬបង្ហាញសកម្មភាពដែលបន្ត/ធ្វើម្តងហើយម្តងទៀត៖

- *kanti* (ច្រៀង) → _kant**ad**o_ (ការច្រៀង)
- *suferi* (រងទុក្ខ) → _sufer**ad**o_ (ការរងទុក្ខ)

សម្រាប់សកម្មភាពបន្ត៖

- *rigardi* (មើល) → _rigard**ad**i_ (មើលបន្ត / សម្លឹង)
- *demandi* (សួរ) → _demand**ad**i_ (សួរម្តងហើយម្តងទៀត)
- *informo* (ព័ត៌មាន) → _inform**ad**o_ (ការផ្សព្វផ្សាយព័ត៌មាន)

# បច្ច័យ *-ar*

បច្ច័យ _**-ar**_ បង្ហាញក្រុម ឬសំណុំដែលមើលជារួម៖

- *arbo* (ដើមឈើ) → _arb**ar**o_ (ព្រៃ)
- *vagono* (ទូរថភ្លើង) → _vagon**ar**o_ (រថភ្លើង)
- *vorto* (ពាក្យ) → _vort**ar**o_ (វចនានុក្រម)

# បច្ច័យ *-um*

បច្ច័យ _**-um**_ គ្មានន័យថេរតែមួយទេ។ ត្រូវរៀនពាក្យដែលមានវាជាពាក្យរៀងៗខ្លួន។

- *plena* (ពេញ) → _plen**um**i_ (បំពេញ / សម្រេច)
- *proksima* (ជិត) → _proksim**um**e_ (ប្រហែល)
- *suno* (ព្រះអាទិត្យ) → _sun**um**i_ (ហាលថ្ងៃ)
- _malvarm**um**i_ – ផ្តាសាយ

**💡 ចងចាំ:** *-um* ដូចជាប្រអប់ពិសេស។ កុំទាយអត្ថន័យពេក ត្រូវពិនិត្យពាក្យពេញ។
