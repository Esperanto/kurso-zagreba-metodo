# ຮູບເງື່ອນໄຂ

ຮູບເງື່ອນໄຂສະແດງດ້ວຍທ້າຍ *__-us__*:

- *mi kred__us__* – ຂ້ອຍຈະເຊື່ອ
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* – ຖ້າຂ້ອຍສຸຂະພາບດີ, ຂ້ອຍຈະມີຄວາມສຸກຫຼາຍ.

# *Kvazaŭ*

*kvazaŭ* ໃຊ້ເປັນສັນທານ “ຄືກັບວ່າ”, ແລະມັກຕາມດ້ວຍຮູບເງື່ອນໄຂ:

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* – ເຈົ້ານັ່ງຢູ່ນັ້ນຄືກັບວ່າເຈົ້າເປັນຄົນຮັ່ງມີ.

ບາງຄັ້ງສາມາດລະບາງຄຳໄດ້:

- *Vi sidas tie __kvazaŭ__ riĉulo.* – ເຈົ້ານັ່ງຢູ່ນັ້ນຄືຄົນຮັ່ງມີ.

# ປະຕິຍັດ *-ad*

*-ad* ສ້າງຄຳນາມຈາກກິລິຍາ:

- *kanti* (ຮ້ອງເພງ) → *kant__ad__o* (ການຮ້ອງເພງ)
- *suferi* (ທົນທຸກ) → *sufer__ad__o* (ການທົນທຸກ)

ມັນຍັງສະແດງການກະທຳຕໍ່ເນື່ອງ ຫຼືຊ້ຳໆ:

- *rigardi* (ເບິ່ງ) → *rigard__ad__i* (ເບິ່ງຕໍ່ໆໄປ)
- *demandi* (ຖາມ) → *demand__ad__i* (ຖາມຊ້ຳໆ)
- *informo* (ຂໍ້ມູນ) → *inform__ad__o* (ການປະຊາສຳພັນ)

# ປະຕິຍັດ *-ar*

*-ar* ສະແດງກຸ່ມຫຼືຊຸດຂອງສິ່ງຕ່າງໆ:

- *arbo* (ຕົ້ນໄມ້) → *arb__ar__o* (ປ່າໄມ້)
- *vagono* (ຕູ້ລົດໄຟ) → *vagon__ar__o* (ລົດໄຟ)
- *vorto* (ຄຳ) → *vort__ar__o* (ວັດຈະນານຸກົມ)

# ປະຕິຍັດ *-um*

*-um* ແມ່ນປະຕິຍັດທີ່ບໍ່ມີຄວາມໝາຍຕາຍຕົວ. ຕ້ອງຮຽນເປັນຄຳໆ:

- *plena* (ເຕັມ) → *plen__um__i* (ປະຕິບັດໃຫ້ສຳເລັດ)
- *proksima* (ໃກ້) → *proksim__um__e* (ປະມານ)
- *suno* (ດວງອາທິດ) → *sun__um__i* (ອາບແດດ)
- *malvarm__um__i* – ເປັນຫວັດ
