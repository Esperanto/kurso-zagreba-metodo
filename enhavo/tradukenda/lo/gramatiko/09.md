# ຮູບເງື່ອນໄຂ

ຮູບເງື່ອນໄຂສະແດງດ້ວຍທ້າຍ _**-us**_:

- _mi kred**us**_ – ຂ້ອຍຈະເຊື່ອ
- _Se mi est**us** sana, mi est**us** tre feliĉa._ – ຖ້າຂ້ອຍສຸຂະພາບດີ, ຂ້ອຍຈະມີຄວາມສຸກຫຼາຍ.

# *Kvazaŭ*

*kvazaŭ* ໃຊ້ເປັນສັນທານ “ຄືກັບວ່າ”, ແລະມັກຕາມດ້ວຍຮູບເງື່ອນໄຂ:

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ – ເຈົ້ານັ່ງຢູ່ນັ້ນຄືກັບວ່າເຈົ້າເປັນຄົນຮັ່ງມີ.

ບາງຄັ້ງສາມາດລະບາງຄຳໄດ້:

- _Vi sidas tie **kvazaŭ** riĉulo._ – ເຈົ້ານັ່ງຢູ່ນັ້ນຄືຄົນຮັ່ງມີ.

# ປະຕິຍັດ *-ad*

*-ad* ສ້າງຄຳນາມຈາກກິລິຍາ:

- *kanti* (ຮ້ອງເພງ) → _kant**ad**o_ (ການຮ້ອງເພງ)
- *suferi* (ທົນທຸກ) → _sufer**ad**o_ (ການທົນທຸກ)

ມັນຍັງສະແດງການກະທຳຕໍ່ເນື່ອງ ຫຼືຊ້ຳໆ:

- *rigardi* (ເບິ່ງ) → _rigard**ad**i_ (ເບິ່ງຕໍ່ໆໄປ)
- *demandi* (ຖາມ) → _demand**ad**i_ (ຖາມຊ້ຳໆ)
- *informo* (ຂໍ້ມູນ) → _inform**ad**o_ (ການປະຊາສຳພັນ)

# ປະຕິຍັດ *-ar*

*-ar* ສະແດງກຸ່ມຫຼືຊຸດຂອງສິ່ງຕ່າງໆ:

- *arbo* (ຕົ້ນໄມ້) → _arb**ar**o_ (ປ່າໄມ້)
- *vagono* (ຕູ້ລົດໄຟ) → _vagon**ar**o_ (ລົດໄຟ)
- *vorto* (ຄຳ) → _vort**ar**o_ (ວັດຈະນານຸກົມ)

# ປະຕິຍັດ *-um*

*-um* ແມ່ນປະຕິຍັດທີ່ບໍ່ມີຄວາມໝາຍຕາຍຕົວ. ຕ້ອງຮຽນເປັນຄຳໆ:

- *plena* (ເຕັມ) → _plen**um**i_ (ປະຕິບັດໃຫ້ສຳເລັດ)
- *proksima* (ໃກ້) → _proksim**um**e_ (ປະມານ)
- *suno* (ດວງອາທິດ) → _sun**um**i_ (ອາບແດດ)
- _malvarm**um**i_ – ເປັນຫວັດ
