# Priesagos *-et-* ir *-eg-*

*__-et-__* mažina, o *__-eg-__* didina reikšmę:

- *libro* (knyga) → *libr__et__o* (knygelė)
- *domo* (namas) → *dom__eg__o* (didžiulis namas)
- *varma* (šiltas) → *varm__eg__a* (labai karštas)

Lietuviškai dažnai vartojame mažybines priesagas, todėl *-et-* gali būti ypač natūralus.

# Priesaga *-iĝ-*

*__-iĝ-__* reiškia „tapti“ arba „pasidaryti“:

- *ruĝa* (raudonas) → *ruĝ__iĝ__i* (parausti)
- *sana* (sveikas) → *san__iĝ__i* (pasveikti)

# Priesaga *-ig-*

*__-ig-__* reiškia „padaryti, sukelti“:

- *pura* (švarus) → *pur__ig__i* (išvalyti)
- *forta* (stiprus) → *fort__ig__i* (sustiprinti)

**⚠️ Dėmesio:** *-iĝ-* = pats tampa, *-ig-* = kažkas padaro. *ruĝiĝi* (parausti) nėra tas pats kaip *ruĝigi* (padaryti raudoną).
