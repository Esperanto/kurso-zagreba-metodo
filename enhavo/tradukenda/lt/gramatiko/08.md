# Tariamoji nuosaka: *-us*

Galūnė *__-us__* reiškia sąlygą, mandagų norą arba įsivaizduojamą veiksmą:

- *Mi irus, se mi havus tempon.* (Eičičiau, jei turėčiau laiko.)
- *Ĉu vi volus trinki teon?* (Ar norėtum gerti arbatos?)

Lietuvių kalboje tam turime formas „eičiau“, „turėčiau“, o esperanto visur vartoja *-us*.

# Priesaga *-ebl-*

*__-ebl-__* reiškia „galimas“:

- *trinki* (gerti) → *trink__ebl__a* (geriamas)
- *legi* (skaityti) → *leg__ebl__a* (įskaitomas)

# Priesaga *-ind-*

*__-ind-__* reiškia „vertas“:

- *legi* (skaityti) → *leg__ind__a* (vertas skaityti)
- *havi* (turėti) → *hav__ind__a* (vertas turėti)
