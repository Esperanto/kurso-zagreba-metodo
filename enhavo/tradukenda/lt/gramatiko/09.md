# Priesaga *-aĵ-*

*__-aĵ-__* paverčia savybę ar veiksmą konkrečiu dalyku:

- *bela* (gražus) → *bel__aĵ__o* (gražus dalykas)
- *manĝi* (valgyti) → *manĝ__aĵ__o* (maistas)
- *nova* (naujas) → *nov__aĵ__o* (naujiena)

# Priesaga *-ec-*

*__-ec-__* reiškia abstrakčią savybę:

- *bela* (gražus) → *bel__ec__o* (grožis)
- *amika* (draugiškas) → *amik__ec__o* (draugiškumas)

**💡 Patarimas:** *-aĵ-* dažniau yra daiktas, *-ec-* - savybė.

# Priesaga *-ar-*

*__-ar-__* reiškia grupę arba rinkinį:

- *vorto* (žodis) → *vort__ar__o* (žodynas)
- *homo* (žmogus) → *hom__ar__o* (žmonija)
