# Tariamoji nuosaka

Ji reiškiama veiksmažodžio galūne *__-us__*. Ji tinka hipotetinėms ar sąlyginėms situacijoms:

- *mi kred__us__* - aš tikėčiau / manyčiau
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* - Jei būčiau sveikas, būčiau labai laimingas.

**⚠️ Dėmesio:** abiejose sakinio dalyse vartojama *-us*: ir „jei“ dalyje, ir pagrindinėje dalyje.

**💡 Patarimas:** dabar veiksmažodžių sistema pilna: *-i* (bendratis), *-as / -is / -os* (trys laikai), *-u* (paliepimas / pageidavimas), *-us* (sąlyga).

# *Kvazaŭ*

vartojamas kaip jungtukas, paprastai kartu su tariamąja nuosaka:

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* - Tu ten sėdi taip, tarsi būtum turtuolis.

Galima vartoti ir trumpiau:

- *Vi sidas tie __kvazaŭ__ riĉulo.* - Tu ten sėdi kaip turtuolis.

# Priesaga *__-ad__*

iš veiksmažodžio sudaro daiktavardį, kuris pavadina patį veiksmą:

- *kanti* (dainuoti) → *kant__ad__o* - dainavimas
- *suferi* (kentėti) → *sufer__ad__o* - kentėjimas

Ji gali rodyti ir ilgalaikį arba pasikartojantį veiksmą:

- *rigardi* (žiūrėti) → *rigard__ad__i* - ilgai žiūrėti
- *demandi* (klausti) → *demand__ad__i* - kartotinai klausinėti
- *informo* (informacija) → *inform__ad__o* - informavimas

# Priesaga *__-ar__*

rodo daiktų rinkinį kaip vieną visumą:

- *arbo* (medis) → *arb__ar__o* - miškas
- *vagono* (vagonas) → *vagon__ar__o* - traukinys
- *vorto* (žodis) → *vort__ar__o* - žodynas

**💡 Patarimas:** *-ar* sudeda atskirus dalykus į grupę:

- *arbo* (medis) → *arbaro* (miškas)

# Priesaga *__-um__*

yra laisvos reikšmės priesaga be vienos apibrėžtos funkcijos:

- *plena* (pilnas) → *plen__um__i* - įvykdyti
- *proksima* (artimas) → *proksim__um__e* - apytiksliai
- *suno* (saulė) → *sun__um__i* - degintis saulėje
- *malvarm__um__i* - persišaldyti

**💡 Patarimas:** *-um* žodžius mokykitės po vieną; tai panašu į prielinksnį *je* ([6 pamoka](/lt/06/gramatiko/)).
