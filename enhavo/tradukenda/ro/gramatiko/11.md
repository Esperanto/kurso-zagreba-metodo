# Sufixul *-il*

unealtă, instrument:

- *tranĉ__il__o* – cuțit
- *hak__il__o* – topor
- *ŝlos__il__o* – cheie
- *timig__il__o* – sperietoare

**💡 Truc de memorare:** *-il* este mijlocul sau instrumentul pentru acțiune: *tranĉi* (a tăia) → *tranĉilo* (cuțit).

**💡 Pont:** *manĝilaro* combină două sufixe deja cunoscute: *-il* (instrument) + *-ar* (colecție) = întregul set de instrumente pentru mâncat, adică tacâmurile.
