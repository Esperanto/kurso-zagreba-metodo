# Podmieňovací spôsob

Podmieňovací spôsob má slovesnú koncovku _**-us**_:

- _mi kred**us**_ – veril by som
- _Se mi est**us** sana, mi est**us** tre feliĉa._ – Keby som bol zdravý, bol by som veľmi šťastný.

**💡 Pomôcka:** _**-us**_ je tvar pre "by": robil by som, bol by, išli by.

# *Kvazaŭ*

*Kvazaŭ* znamená "akoby". Ako spojka býva často so slovesom v podmieňovacom spôsobe:

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ – Sedíš tam, akoby si bol boháč.

Sloveso môžeme vynechať:

- _Vi sidas tie **kvazaŭ** riĉulo._ – Sedíš tam ako boháč.

# Prípona *-ad*

Prípona _**-ad**_ sa používa na tvorbu podstatných mien zo slovies:

- *kanti* (spievať) → _kant**ad**o_ (spev)
- *suferi* (trpieť) → _sufer**ad**o_ (utrpenie)

Môže tiež označovať trvajúci alebo opakovaný dej:

- *rigardi* (pozerať sa) → _rigard**ad**i_ (dlho hľadieť)
- *demandi* (pýtať sa) → _demand**ad**i_ (vypytovať sa)
- *informo* (informácia) → _inform**ad**o_ (informovanie)

**⚠️ Pozor:** _**-ad**_ nie je iba slovenská koncovka "-nie". Často pridáva význam trvania alebo opakovania.

# Prípona *-ar*

Prípona _**-ar**_ označuje súbor alebo skupinu vecí rovnakého druhu:

- *arbo* (strom) → _arb**ar**o_ (les)
- *vagono* (vozeň) → _vagon**ar**o_ (vlak)
- *vorto* (slovo) → _vort**ar**o_ (slovník)

# Prípona *-um*

Prípona _**-um**_ nemá pevný význam. Slová s ňou súvisia so základom, ale význam sa často treba naučiť osobitne:

- *plena* (plný) → _plen**um**i_ (splniť)
- *proksima* (blízky) → _proksim**um**e_ (približne)
- *suno* (slnko) → _sun**um**i_ (opaľovať sa)
- *malvarma* (studený) → _malvarm**um**i_ (prechladnúť)

**💡 Pomôcka:** pri _**-um**_ nečakaj úplne mechanický význam. Ber ho ako signál, že slovo sa oplatí naučiť zvlášť.
