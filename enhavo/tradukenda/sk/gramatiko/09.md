# Podmieňovací spôsob

Podmieňovací spôsob má slovesnú koncovku *__-us__*:

- *mi kred__us__* – veril by som
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* – Keby som bol zdravý, bol by som veľmi šťastný.

**💡 Pomôcka:** *__-us__* je tvar pre "by": robil by som, bol by, išli by.

# *Kvazaŭ*

*Kvazaŭ* znamená "akoby". Ako spojka býva často so slovesom v podmieňovacom spôsobe:

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* – Sedíš tam, akoby si bol boháč.

Sloveso môžeme vynechať:

- *Vi sidas tie __kvazaŭ__ riĉulo.* – Sedíš tam ako boháč.

# Prípona *-ad*

Prípona *__-ad__* sa používa na tvorbu podstatných mien zo slovies:

- *kanti* (spievať) → *kant__ad__o* (spev)
- *suferi* (trpieť) → *sufer__ad__o* (utrpenie)

Môže tiež označovať trvajúci alebo opakovaný dej:

- *rigardi* (pozerať sa) → *rigard__ad__i* (dlho hľadieť)
- *demandi* (pýtať sa) → *demand__ad__i* (vypytovať sa)
- *informo* (informácia) → *inform__ad__o* (informovanie)

**⚠️ Pozor:** *__-ad__* nie je iba slovenská koncovka "-nie". Často pridáva význam trvania alebo opakovania.

# Prípona *-ar*

Prípona *__-ar__* označuje súbor alebo skupinu vecí rovnakého druhu:

- *arbo* (strom) → *arb__ar__o* (les)
- *vagono* (vozeň) → *vagon__ar__o* (vlak)
- *vorto* (slovo) → *vort__ar__o* (slovník)

# Prípona *-um*

Prípona *__-um__* nemá pevný význam. Slová s ňou súvisia so základom, ale význam sa často treba naučiť osobitne:

- *plena* (plný) → *plen__um__i* (splniť)
- *proksima* (blízky) → *proksim__um__e* (približne)
- *suno* (slnko) → *sun__um__i* (opaľovať sa)
- *malvarma* (studený) → *malvarm__um__i* (prechladnúť)

**💡 Pomôcka:** pri *__-um__* nečakaj úplne mechanický význam. Ber ho ako signál, že slovo sa oplatí naučiť zvlášť.
