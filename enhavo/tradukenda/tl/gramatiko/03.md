# Mga Pang-abay

Ang hinangong pang-abay ay ginagawa sa hulaping _**-e**_. Kumpleto na ang sistema ng patinig: *-o* para sa pangngalan, *-a* para sa pang-uri, *-e* para sa pang-abay.

- _bel**e**_ - nang maganda
- _fort**e**_ - nang malakas
- _rapid**a** aŭto_ - mabilis na kotse
  - _veturi rapid**e**_ - bumiyahe nang mabilis

# Ang Pang-ukol na *per* at *kun*

Pareho silang maaaring isaling "kasama/gamit", ngunit magkaiba ang tungkulin.

## *Per* - sa pamamagitan ng, gamit ang

- _Manĝi **per** kulero_ - kumain gamit ang kutsara
- _Ŝi kantis **per** tre bela voĉo._ - Kumanta siya sa pamamagitan ng napakagandang boses.

## *Kun* - kasama

- _Mi iris **kun** la amiko._ - Pumunta ako kasama ang kaibigan.
- _Mi parolos **kun** li._ - Makikipag-usap ako sa kanya.

**💡 Tandaan:** *kun* ay kasama; *per* ay kasangkapan o paraan.

# Ang Pang-ukol na *post*

Ang *post* ay "pagkatapos" kapag panahon ang tinutukoy:

- _Li venis **post** mi._ - Dumating siya pagkatapos ko.
- _**post** du horoj_ - makalipas ang dalawang oras
- _Li venos **post** tri horoj._ - Darating siya makalipas ang tatlong oras.
- *poste* - pagkatapos, kalaunan

# Ang Pang-ukol na *malantaŭ*

Ang *malantaŭ* ay "sa likod" kapag lugar ang tinutukoy:

- _Li venis **malantaŭ** mi._ - Dumating siya sa likod ko.

**⚠️ Babala:** tama ang diin: _mal**an**taŭ_. Ito ay *mal-* + *antaŭ* ("sa harap"), kaya literal itong kabaligtaran ng "sa harap".

# Ang Hulaping _**-ul**_

Tumutukoy ito sa taong may isang katangian:

- _grand**ul**o_ - malaking tao
- _malbon**ul**o_ - masamang tao
- _bel**ul**ino_ - magandang babae

# Ang Hulaping _**-ej**_

Gumagawa ito ng salitang para sa lugar kung saan karaniwang nangyayari ang isang bagay:

- _lern**ej**o_ - paaralan
- _kuir**ej**o_ - kusina (*kuiri* - magluto)
- _labor**ej**o_ - lugar ng trabaho

**💡 Tandaan:** ginagawang lugar ng *-ej* ang gawain:

- *lerni* (matuto) -> *lernejo* (paaralan)
- *kuiri* (magluto) -> *kuirejo* (kusina)

# Ang Hulaping _**-ebl**_

Nagpapakita ito ng posibilidad, katulad ng "-able/-ible":

- _manĝ**ebl**a_ - nakakain
- _vid**ebl**a_ - nakikita
- _kompren**ebl**e_ - nauunawaan, siyempre
- _**ebl**e_ - marahil, siguro
- _mal**ebl**a_ - imposible

# Pautos

Ang hulapi ng utos ay _**-u**_:

- _Manĝ**u**!_ - Kumain ka!
- _Ir**u**!_ - Pumunta ka!

Maaari rin itong gamitin na may simuno:

- _Li lern**u**!_ - Hayaan siyang matuto!
- _Ni vid**u**._ - Tingnan natin.
