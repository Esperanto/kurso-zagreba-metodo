# Mga Pang-abay

Ang hinangong pang-abay ay ginagawa sa hulaping *__-e__*. Kumpleto na ang sistema ng patinig: *-o* para sa pangngalan, *-a* para sa pang-uri, *-e* para sa pang-abay.

- *bel__e__* - nang maganda
- *fort__e__* - nang malakas
- *rapid__a__ aŭto* - mabilis na kotse
	- *veturi rapid__e__* - bumiyahe nang mabilis

# Ang Pang-ukol na *per* at *kun*

Pareho silang maaaring isaling "kasama/gamit", ngunit magkaiba ang tungkulin.

## *Per* - sa pamamagitan ng, gamit ang

- *Manĝi __per__ kulero* - kumain gamit ang kutsara
- *Ŝi kantis __per__ tre bela voĉo.* - Kumanta siya sa pamamagitan ng napakagandang boses.

## *Kun* - kasama

- *Mi iris __kun__ la amiko.* - Pumunta ako kasama ang kaibigan.
- *Mi parolos __kun__ li.* - Makikipag-usap ako sa kanya.

**💡 Tandaan:** *kun* ay kasama; *per* ay kasangkapan o paraan.

# Ang Pang-ukol na *post*

Ang *post* ay "pagkatapos" kapag panahon ang tinutukoy:

- *Li venis __post__ mi.* - Dumating siya pagkatapos ko.
- *__post__ du horoj* - makalipas ang dalawang oras
- *Li venos __post__ tri horoj.* - Darating siya makalipas ang tatlong oras.
- *poste* - pagkatapos, kalaunan

# Ang Pang-ukol na *malantaŭ*

Ang *malantaŭ* ay "sa likod" kapag lugar ang tinutukoy:

- *Li venis __malantaŭ__ mi.* - Dumating siya sa likod ko.

**⚠️ Babala:** tama ang diin: *mal__an__taŭ*. Ito ay *mal-* + *antaŭ* ("sa harap"), kaya literal itong kabaligtaran ng "sa harap".

# Ang Hulaping *__-ul__*

Tumutukoy ito sa taong may isang katangian:

- *grand__ul__o* - malaking tao
- *malbon__ul__o* - masamang tao
- *bel__ul__ino* - magandang babae

# Ang Hulaping *__-ej__*

Gumagawa ito ng salitang para sa lugar kung saan karaniwang nangyayari ang isang bagay:

- *lern__ej__o* - paaralan
- *kuir__ej__o* - kusina (*kuiri* - magluto)
- *labor__ej__o* - lugar ng trabaho

**💡 Tandaan:** ginagawang lugar ng *-ej* ang gawain:

- *lerni* (matuto) -> *lernejo* (paaralan)
- *kuiri* (magluto) -> *kuirejo* (kusina)

# Ang Hulaping *__-ebl__*

Nagpapakita ito ng posibilidad, katulad ng "-able/-ible":

- *manĝ__ebl__a* - nakakain
- *vid__ebl__a* - nakikita
- *kompren__ebl__e* - nauunawaan, siyempre
- *__ebl__e* - marahil, siguro
- *mal__ebl__a* - imposible

# Pautos

Ang hulapi ng utos ay *__-u__*:

- *Manĝ__u__!* - Kumain ka!
- *Ir__u__!* - Pumunta ka!

Maaari rin itong gamitin na may simuno:

- *Li lern__u__!* - Hayaan siyang matuto!
- *Ni vid__u__.* - Tingnan natin.
