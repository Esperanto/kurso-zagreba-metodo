# Ang Pang-ukol na *da*

Ginagamit ang *da* sa pahayag ng timbang, sukat o dami:

- *kilogramo __da__ sukero* - isang kilo ng asukal
- *glaso __da__ akvo* - isang baso ng tubig
- *multe __da__ ideoj* - maraming ideya

**⚠️ Babala:** pagkatapos ng salitang nagsasabi ng dami, gamitin ang *da*, hindi *de*. *glaso da akvo* = baso ng tubig.

# Ang Hulaping *__-ig__*

Ang hulaping ito ay "magpa-/magdulot na maging":

- *bel__ig__i* - pagandahin
- *malplen__ig__i* - alisan ng laman
- *san__ig__i* - pagalingin
- *mort__ig__i* - pumatay

**⚠️ Babala:** huwag ipagkamali ang *-ig* at *-iĝ* ([aralin 6](/tl/06/gramatiko/)). *-ig* = magdulot ng pagbabago; *-iĝ* = maging o magbago ang simuno mismo.

# Ang Hulaping *__-aĉ__*

Ipinapakita nito ang mababang pagtingin o pangit na kalidad:

- *hund__aĉ__o* - asong hamak
- *knab__aĉ__o* - batang pasaway
- *skrib__aĉ__i* - magsulat nang pangit
