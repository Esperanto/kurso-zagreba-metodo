# Kondisyonal

Ipinapahayag ang kondisyonal sa hulaping _**-us**_. Ginagamit ito para sa haka, nais, o pangyayaring hindi tiyak:

- _mi kred**us**_ - maniniwala sana ako
- _Se mi est**us** sana, mi est**us** tre feliĉa._ - Kung ako ay malusog, magiging napakasaya ko.

**⚠️ Babala:** parehong bahagi ng pangungusap ay gumagamit ng *-us* kapag kondisyonal ang diwa: ang sugnay na "kung" at ang pangunahing sugnay.

**💡 Tandaan:** kumpleto na ang sistema ng pandiwa: *-i* (pawatas), *-as/-is/-os* (tatlong panahon), *-u* (utos/hiling), *-us* (kondisyonal).

# *Kvazaŭ*

Ginagamit ang *kvazaŭ* bilang pangatnig na "parang/tila", karaniwang kasama ang kondisyonal:

- _Vi sidas tie **kvazaŭ** vi estus riĉulo._ - Nakaupo ka roon na para kang mayaman.

Maaari rin itong gamitin nang pinaikli:

- _Vi sidas tie **kvazaŭ** riĉulo._ - Nakaupo ka roon na parang mayaman.

# Ang Hulaping _**-ad**_

Gumagawa ito ng pangngalan mula sa pandiwa, para pangalanan ang kilos mismo:

- *kanti* (kumanta) -> _kant**ad**o_ - pagkanta
- *suferi* (magdusa) -> _sufer**ad**o_ - pagdurusa

Nagpapahiwatig din ito ng tuloy-tuloy o paulit-ulit na kilos:

- *rigardi* (tumingin) -> _rigard**ad**i_ - tumingin nang matagal
- *demandi* (magtanong) -> _demand**ad**i_ - paulit-ulit na magtanong
- *informo* (impormasyon) -> _inform**ad**o_ - pagpapabatid

# Ang Hulaping _**-ar**_

Ipinapakita nito ang kalipunan ng mga bagay bilang isang buo:

- *arbo* (puno) -> _arb**ar**o_ - kagubatan
- *vagono* (bagon) -> _vagon**ar**o_ - tren
- *vorto* (salita) -> _vort**ar**o_ - talasalitaan

**💡 Tandaan:** pinagbubuklod ng *-ar* ang hiwa-hiwalay na bagay:

- *arbo* (puno) -> *arbaro* (kagubatan)

# Ang Hulaping _**-um**_

Ang *-um* ay hulaping walang tiyak na kahulugan. Kailangang matutuhan ang bawat salita:

- *plena* (puno) -> _plen**um**i_ - tuparin
- *proksima* (malapit) -> _proksim**um**e_ - humigit-kumulang
- *suno* (araw) -> _sun**um**i_ - magpaaraw
- _malvarm**um**i_ - sipunin

**💡 Tandaan:** ang *-um* ay parang joker sa mga hulapi, tulad ng *je* ([aralin 6](/tl/06/gramatiko/)) sa mga pang-ukol.
