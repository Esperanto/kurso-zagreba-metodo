# Kondisyonal

Ipinapahayag ang kondisyonal sa hulaping *__-us__*. Ginagamit ito para sa haka, nais, o pangyayaring hindi tiyak:

- *mi kred__us__* - maniniwala sana ako
- *Se mi est__us__ sana, mi est__us__ tre feliĉa.* - Kung ako ay malusog, magiging napakasaya ko.

**⚠️ Babala:** parehong bahagi ng pangungusap ay gumagamit ng *-us* kapag kondisyonal ang diwa: ang sugnay na "kung" at ang pangunahing sugnay.

**💡 Tandaan:** kumpleto na ang sistema ng pandiwa: *-i* (pawatas), *-as/-is/-os* (tatlong panahon), *-u* (utos/hiling), *-us* (kondisyonal).

# *Kvazaŭ*

Ginagamit ang *kvazaŭ* bilang pangatnig na "parang/tila", karaniwang kasama ang kondisyonal:

- *Vi sidas tie __kvazaŭ__ vi estus riĉulo.* - Nakaupo ka roon na para kang mayaman.

Maaari rin itong gamitin nang pinaikli:

- *Vi sidas tie __kvazaŭ__ riĉulo.* - Nakaupo ka roon na parang mayaman.

# Ang Hulaping *__-ad__*

Gumagawa ito ng pangngalan mula sa pandiwa, para pangalanan ang kilos mismo:

- *kanti* (kumanta) -> *kant__ad__o* - pagkanta
- *suferi* (magdusa) -> *sufer__ad__o* - pagdurusa

Nagpapahiwatig din ito ng tuloy-tuloy o paulit-ulit na kilos:

- *rigardi* (tumingin) -> *rigard__ad__i* - tumingin nang matagal
- *demandi* (magtanong) -> *demand__ad__i* - paulit-ulit na magtanong
- *informo* (impormasyon) -> *inform__ad__o* - pagpapabatid

# Ang Hulaping *__-ar__*

Ipinapakita nito ang kalipunan ng mga bagay bilang isang buo:

- *arbo* (puno) -> *arb__ar__o* - kagubatan
- *vagono* (bagon) -> *vagon__ar__o* - tren
- *vorto* (salita) -> *vort__ar__o* - talasalitaan

**💡 Tandaan:** pinagbubuklod ng *-ar* ang hiwa-hiwalay na bagay:

- *arbo* (puno) -> *arbaro* (kagubatan)

# Ang Hulaping *__-um__*

Ang *-um* ay hulaping walang tiyak na kahulugan. Kailangang matutuhan ang bawat salita:

- *plena* (puno) -> *plen__um__i* - tuparin
- *proksima* (malapit) -> *proksim__um__e* - humigit-kumulang
- *suno* (araw) -> *sun__um__i* - magpaaraw
- *malvarm__um__i* - sipunin

**💡 Tandaan:** ang *-um* ay parang joker sa mga hulapi, tulad ng *je* ([aralin 6](/tl/06/gramatiko/)) sa mga pang-ukol.
