# Ang Hulaping _**-an**_

Tumutukoy ito sa kasapi o taong kabilang sa isang grupo:

- _klub**an**o_ - kasapi ng klub
- _amerik**an**o_ - Amerikano
- _samland**an**o_ - kababayan
- _samide**an**o_ - kasamahan sa parehong ideya, lalo na kapwa Esperantista

# Ang Hulaping _**-ec**_

Gumagawa ito ng abstraktong pangngalan: kalagayan o katangian.

- _bel**ec**o_ - kagandahan
- _varm**ec**o_ - init
- _simpl**ec**o_ - kasimplihan

**💡 Tandaan:** ginagawang katangian ng *-ec* ang pang-uri:

- *bela* (maganda) -> *beleco* (kagandahan)

# Ang Hulaping _**-uj**_

Gumagawa ito ng pangalan ng lalagyan, bansa, o halaman/puno:

- _manĝ**uj**o_ - pakainan
- _cindr**uj**o_ - lalagyan ng abo
- _Franc**uj**o_ - Pransiya
- _Aŭstr**uj**o_ - Awstriya
- *pomo* (mansanas) -> _pom**uj**o_ - puno ng mansanas
