# Ang Hulaping _**-il**_

Gumagawa ito ng salita para sa kasangkapan, gamit, o paraan:

- _tranĉ**il**o_ - kutsilyo
- _hak**il**o_ - palakol
- _ŝlos**il**o_ - susi
- _timig**il**o_ - panakot

**💡 Tandaan:** pinapangalanan ng *-il* ang kasangkapan para sa kilos:

- *tranĉi* (maghiwa) -> *tranĉilo* (kutsilyo)
- *ŝlosi* (magsusi) -> *ŝlosilo* (susi)
