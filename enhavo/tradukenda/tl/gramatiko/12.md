# *Ju ... des*

Ang *ju pli ... des pli* ay katumbas ng "habang mas ... mas ...":

- *Ju pli longe, des pli bone.* - Habang mas matagal, mas mabuti.

**💡 Tandaan:** *ju* sa unang bahagi, *des* sa ikalawa. Magkapareha ang dalawang salitang ito.

# *Ajn*

Ginagawa ng *ajn* na di-tiyak o pangkalahatan ang salita:

- *kiom __ajn__* - gaano man karami

**💡 Tandaan:** binubuksan ng *ajn* ang isang [tabelvorto](/tl/tabelvortoj/) sa kahulugang "kahit":

- *kiom* (gaano karami) -> *kiom ajn* (gaano man karami)

# Ang Unlaping *__dis-__*

Nagpapakita ito ng paghihiwalay o pagkalat:

- *__dis__ĵeti* - ikalat
- *__dis__sendi* - ipalaganap

# Ang Hulaping *__-on__*

Nagpapakita ito ng bahagi o fraction:

- *du__on__o* - kalahati
- *tri__on__o* - ikatlo
- *kvar__on__o* - sangkapat
- *ses__on__o* - ikaanim

**💡 Tandaan:** ginagawa ng *-on* na bahagi ang numero:

- *du* (dalawa) -> *duono* (kalahati)
- *kvar* (apat) -> *kvarono* (sangkapat)
